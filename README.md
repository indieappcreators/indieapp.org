# indieapp.org

A static site built with Clovie.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

- `views/` - HTML templates
- `scripts/` - JavaScript files
- `styles/` - SCSS files
- `assets/` - Static assets
- `dist/` - Build output
